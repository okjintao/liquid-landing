# Color System

Our color system is designed to create a cohesive, cyber-aquatic aesthetic that reflects the Liquid brand identity. The palette is built around a core set of colors extracted from our brand imagery, with carefully selected opacity variants for different use cases.

## Primary Colors

| Color Name           | Hex       | Usage                            |
| -------------------- | --------- | -------------------------------- |
| **Liquid Glow Blue** | `#00CFFF` | Accents, buttons, highlight text |
| **Deep Ocean Blue**  | `#041B2D` | Background base                  |
| **Dark Teal**        | `#073346` | Card backgrounds, nav bars       |
| **Slate Blue**       | `#0E4B66` | Section dividers, borders        |
| **Soft Cyan**        | `#57D4F5` | Hover states, gradients          |

## Neutral Colors

| Color Name          | Hex       | Usage                         |
| ------------------- | --------- | ----------------------------- |
| **Off White**       | `#E4F8FF` | Body text, headers            |
| **Muted Blue Gray** | `#8CAFC3` | Descriptions, labels          |
| **Smoke Black**     | `#0A0F12` | Modal overlays, deep contrast |

## CSS Variables

All colors are available as CSS variables for consistent usage across the application:

```css
:root {
  /* Primary Colors */
  --color-liquid-glow: #00CFFF;
  --color-deep-ocean: #041B2D;
  --color-dark-teal: #073346;
  --color-slate-blue: #0E4B66;
  --color-soft-cyan: #57D4F5;

  /* Neutral Colors */
  --color-off-white: #E4F8FF;
  --color-muted-blue: #8CAFC3;
  --color-smoke-black: #0A0F12;

  /* Opacity Variants */
  --color-liquid-glow-5: rgba(0, 207, 255, 0.05);
  --color-liquid-glow-10: rgba(0, 207, 255, 0.1);
  --color-liquid-glow-30: rgba(0, 207, 255, 0.3);
  --color-dark-teal-90: rgba(7, 51, 70, 0.9);
  --color-smoke-black-95: rgba(10, 15, 18, 0.95);
}
```

## Utility Classes

We provide several utility classes for common color patterns:

- `.gradient-text`: Text with a gradient from Liquid Glow to Soft Cyan
- `.gradient-border`: Border with Slate Blue at 50% opacity
- `.card-bg`: Card background with Dark Teal at 90% opacity
- `.hover-glow`: Hover effect transitioning to Liquid Glow
- `.active-link`: Active link state with Liquid Glow
- `.inactive-link`: Inactive link state with hover effect

## Usage Guidelines

1. **Text Colors**
   - Use Off White (`#E4F8FF`) for primary text
   - Use Muted Blue Gray (`#8CAFC3`) for secondary text
   - Use Liquid Glow (`#00CFFF`) for interactive elements

2. **Backgrounds**
   - Use Deep Ocean (`#041B2D`) as the base background
   - Use Dark Teal (`#073346`) for cards and elevated surfaces
   - Use Smoke Black (`#0A0F12`) for modals and overlays

3. **Interactive Elements**
   - Use Liquid Glow (`#00CFFF`) for primary actions
   - Use Soft Cyan (`#57D4F5`) for hover states
   - Use Slate Blue (`#0E4B66`) for borders and dividers

4. **Gradients**
   - Use Liquid Glow to Soft Cyan for text gradients
   - Use opacity variants for subtle background effects
   - Maintain consistent direction (left to right) for gradients

## Accessibility

All color combinations have been tested to meet WCAG 2.1 AA standards for contrast. When using these colors, ensure:

1. Text maintains a minimum contrast ratio of 4.5:1 with its background
2. Interactive elements maintain a minimum contrast ratio of 3:1
3. Use opacity variants carefully to maintain readability
